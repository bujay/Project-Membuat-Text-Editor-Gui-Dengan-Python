## Requirement

1. saya menggunakan sublime text sebagai text editor atau IDE dan command prompt (windows) 
   sebagai hasil output pemrogrammannya
2. menggunakan versi python 3 ke atas
3. tidak memerlukan penginstalan package atau modul lewat command prompt (windows) 
   karena aplikasi tersebut berbasis tkinter(GUI) yang sudah ada dalam python tersebut

## Cara Menggunakan Source Code menggunakan command prompt (windows)

1. pilih search yang terletak di samping logo windows
2. ketika udah ketemu, langkah selanjutnya ketik command prompt 
   dan ketikan perintah yang dimana lokasi penyimpanan file "project_text_editor" tersebut berada
   misalnya C:\Users\bujai>D:
            D:\>cd python
            D:\>cd project_text_editor
            D:\>python text_editor.py
3. maka hasilnya akan keluar

## Cara Menggunakan aplikasi(exe):

1. buka file explorer
2. cari file "project_text_editor" 
   misalkan D:\>python\project_text_editor
3. ketika udah ketemu file "project_text_editor", 
   langkah selanjutnya pilih file "dist" lalu pilih text_editor, 
   dan pilih text_editor.exe lalu klik 2 kali
   misalkan D:\>python\project_text_editor\dist\text_editor\text_editor.exe
4. maka aplikasi tersebut akan berjalan
